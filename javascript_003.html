자바스크립트 자료형

컴퓨터가 수행할 수 있는 내용을 정리한 출력형태

  • 숫자 : 숫자단독으로사용
  • 문자(글자)"큰따옴표",'작은따옴표'모두 사용가능, 중첩시 교차설정 가능
  • 참/거짓 : 논리형 true, false으로 출력
  • 배열 : 하나의 변수에 여러개의 문자를 첨부하는 형
  • 객체 : 컴퓨터가 인식하고 있는 함수의 구조를 불러오는 형
    대표적으로 날짜 , 시간, 자연수, 소수 이런 내용을 설정
  • 특수형 : null(숫자를 인식못하는 경우), none(문자를 인식 못하는 경우), undefined(자료를 인식 못하는 경우)

연산자 공부하기

  • 산술연산자
    • + : 숫자의 경우 더한다, 문자의 경우 결합한다.
    • - : 숫자의 경우만 있고 수를 뺀다.
    • * : 숫자의 경우만 있고 곱한다.
    • / : 숫자의 경우만 있고 나눈다.
    • % : 숫자의 경우만 있고 나눈 다음 나머지를 뜻한다.
  • 증감연산자
    • ++ : 1씩 더해나간다
    • -- : 1씩 빼나간다
